本文中所使用的 hugo 主题中，series 是默认不在文章开头显示的，那么如何修改才能使 series 的参数和 author/categories/tags 等参数保持相同格式显示在文章内呢？

修改前：

1，

首先，这里以 Visual Studio Code 为例，按照 themes\你的 hugo 主题名称\layouts\partials\… 路径寻找 single.html 文件，在其中找到文章开头信息显示的部分

2,

接下来仿造格式增加一行对应的

          {{ with .GetTerms "series" }}{{ partial "taxonomy/series.html" . }}{{ end }}

代码，并根据代码中的 partial “taxonomy/series.html” 前往当前目录下的 taxonomy 文件夹，新建文件 series.html ，从该文件夹中其他 html 文件复制粘贴格式，并修改自己想要自定义的部分

我主要修改了开头的 class 名和显示的图标

现在文章开头就可以显示 series 的相关信息了
